## About
The **Local Train Ticketing System** is a Java-based ticket booking platform that allows users to book local train tickets digitally.  
Users can select source, destination, journey date, travel class, and number of seats. The system eliminates long queues by enabling online booking, payment, ticket download, and cancellation.

---

## Key Features
- Book local train tickets digitally
- Select journey date, class, and seat count
- User wallet & payment support (UPI, cards, net banking â€“ extensible)
- Download tickets in PDF or digital format
- Cancel tickets within 3 hours of journey
- Admin control for schedules and users
